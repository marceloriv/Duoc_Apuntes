# Unlighthouse

## Objetivo

Auditar rendimiento, SEO y calidad de una web usando Lighthouse desde CLI.

## Comando base

npx unlighthouse --site "url del sitio"

## Ejemplo

npx unlighthouse --site "<https://midominio.cl>"

## Notas

- Reemplaza la URL por tu sitio real.
- Si es primera vez, npx descargara el paquete de forma temporal.
- Ideal para revisiones rapidas antes de despliegue.
