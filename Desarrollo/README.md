# Comandos

Indice rapido de comandos reutilizables.

## Estructura

- General: comandos transversales y herramientas varias.
- Git: flujo de trabajo Git y convenciones.
- Sistema: comandos del sistema operativo.

## Rutas

- General: ./General/
- Git: ./Git/
- Sistema: ./Sistema/

## Convencion sugerida para nuevos archivos

- Nombre corto y claro en PascalCase o kebab-case.
- Un comando por archivo cuando tenga contexto amplio.
- Incluir: objetivo, comando base, ejemplo y notas.
